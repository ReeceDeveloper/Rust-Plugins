# Essentials

The Essentials plugin provides extra functions to improve quality of life while playing Rust.

## Default (Plugin) Configuration

Below is the default plugin configuration settings should you need to reset your configuration.

```json
{
  "(1). Enable the Essentials plugin?": true,
  "(2). Enable the server's whitelist?": false,
  "(3). Announce when the server is saved?": false,
  "(4). Disable F1-give messages?": false,
  "(5). Announce when a player joins the server?": true,
  "(6). Announce when a player leaves the server?": true
}
```

## Default (Language) Configuration

Below is the default language configuration settings should you need to reset your language file.

```json
{
  "WhitelistDenyEvent": "You are not whitelisted on this server.",
  "WorldSaveEvent": "The server is now saving.",
  "PlayerJoinEvent": "{0} has joined the server!",
  "PlayerLeaveEvent": "{0} has left the server!"
}
```

To use colour codes use `[#HEXCODE]Text[/#]` to do such; for example, `[#123ABC]Text[/#]`.
